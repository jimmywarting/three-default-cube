## registerParticleEmitter ()

```
registerParticleEmitter(object, {
  particleObject,
  particleDensity,
  positionBase,
  rotationBase,
  scaleBase,
  positionJitter,
  rotationJitter,
  scaleJitter,
  spawnJitter,
  globalTransforms,
  onCreate,
  onFrame,
  onReset
});
```

Creates new particle emitter. Returns emitter information and reference. See [Particles](/advanced/particles/).
